# 基础类型总览

Lency 的核心基础类型：

- `int`: 整数类型
- `float`: 浮点类型
- `bool`: 布尔类型
- `string`: 字符串类型
- `void`: 无返回值类型（主要用于函数返回）

## 文档分流

- Bool: [bool.md](./bool.md)
- Float: [float.md](./float.md)
- Null 安全: [null-safety.md](./null-safety.md)

## 说明

类型系统在 Rust 主编译器与 Lency 自举编译器之间存在阶段差异。
如需查看当前自举能力，优先参考 [README.md](../README.md) 的“实现状态（2026-03-04）”。
